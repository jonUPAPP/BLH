# BLH

> A smarter conversational assistant designed to provide a more natural and context-aware experience than traditional chatbots.

BLH (Beyond Limits Helper) เป็นระบบผู้ช่วยสนทนาที่ถูกออกแบบมาเพื่อให้สามารถโต้ตอบกับผู้ใช้งานได้อย่างเป็นธรรมชาติ เข้าใจบริบทของบทสนทนา และให้ความช่วยเหลือได้อย่างต่อเนื่อง

แตกต่างจากบอททั่วไปที่มักตอบคำถามแบบแยกเป็นครั้ง ๆ BLH มุ่งเน้นการรักษาความต่อเนื่องของการสนทนา การเชื่อมโยงข้อมูลที่เกี่ยวข้อง และการนำเสนอคำตอบที่เหมาะสมกับสถานการณ์มากขึ้น

---

## ✨ Features

- 🧠 เข้าใจบริบทของการสนทนา
- 💬 ตอบกลับอย่างเป็นธรรมชาติ
- 📚 จัดการและดึงข้อมูลที่เกี่ยวข้องได้อย่างรวดเร็ว
- 🔍 ช่วยค้นหา วิเคราะห์ และสรุปข้อมูล
- 📝 รองรับการจดจำข้อมูลสำคัญระหว่างการสนทนา
- ⚡ ปรับปรุงคุณภาพการตอบกลับอย่างต่อเนื่อง
- 🎯 เน้นประสบการณ์การใช้งานที่ลื่นไหลและเป็นมิตร

---

## 🚀 Why BLH?

BLH ถูกสร้างขึ้นเพื่อแก้ปัญหาที่มักพบในระบบสนทนาแบบเดิม เช่น

- การลืมบริบทของบทสนทนา
- การตอบคำถามแบบไม่ต่อเนื่อง
- การดึงข้อมูลที่เกี่ยวข้องได้ไม่ดีพอ
- การสื่อสารที่ดูเป็นหุ่นยนต์มากเกินไป

เป้าหมายของ BLH คือการทำให้การโต้ตอบกับระบบมีความเป็นธรรมชาติ เข้าใจง่าย และช่วยเหลือผู้ใช้ได้อย่างมีประสิทธิภาพมากขึ้น

---

## 📌 Use Cases

BLH สามารถนำไปประยุกต์ใช้ได้หลากหลายรูปแบบ เช่น

### Learning Assistant
ช่วยค้นคว้า สรุป และอธิบายเนื้อหาต่าง ๆ

### Productivity Assistant
ช่วยวางแผน จัดระเบียบ และบริหารข้อมูล

### Information Assistant
ช่วยค้นหา วิเคราะห์ และจัดเรียงข้อมูล

### Interactive Chat System
ระบบสนทนาเชิงโต้ตอบสำหรับโปรเจกต์หรือบริการต่าง ๆ

## 🛠 Installation

Clone Repository

```bash
git clone https://github.com/jonUPAPP/BLH.git
```

เข้าสู่โฟลเดอร์โปรเจกต์

```bash
cd BLH
```

เปิดไฟล์หลักผ่าน Browser หรือ Web Server

```bash
python -m http.server
```

หรือ

```bash
npx http-server
```

---

## 📖 Quick Start

1. เปิดระบบผ่าน Browser
2. เริ่มต้นการสนทนา
3. ป้อนคำถามหรือข้อมูลที่ต้องการ
4. ให้ BLH ช่วยวิเคราะห์ สรุป หรือให้คำแนะนำ
